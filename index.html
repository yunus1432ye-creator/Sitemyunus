Canlı Ders İspat Ekranı Simülasyonu

Öğrencilerimize formülleri dayatmak yerine, ders esnasında tahtada birlikte inşa ettiğimiz ispat süreçlerinden kesitler:

[EKRAN 1] - Cebirsel İspat: Gauss Toplam Formülü
// Ardışık sayıların toplam formülü ezberlenmeden nasıl çıkarılır?
1 + 2 + 3 + ... + n = [ n × (n + 1) ] / 2
  • 1. Aradığımız toplama S diyoruz: S = 1 + 2 + 3 + ... + n
  • 2. Aynı toplamı altına tamamen ters sırada yazıyoruz: S = n + (n-1) + ... + 1
  • 3. Bu iki satırı alt alta topladığımızda, her kolonun toplamı istisnasız (n+1) olur.
  • 4. Satırda tam n tane kolon olduğu için toplam: 2S = n × (n + 1) halini alır.
  • 5. Buradan S yalnız bırakıldığında formül kendiliğinden ispatlanmış olur.
[EKRAN 2] - Geometrik İspat: İki Kare Farkı
// Özdeşliklerin geometrik alan kesme mantığıyla görselleştirilmesi
a² - b² = (a - b) × (a + b)
  • 1. Bir kenarı "a" olan büyük bir karenin alanını (a²) baz alıyoruz.
  • 2. Köşesinden bir kenarı "b" olan küçük bir kareyi (b²) kesip çıkarıyoruz.
  • 3. Kalan L şeklindeki alanı iki adet dikdörtgen parçaya ayırıyoruz.
  • 4. Bu parçaları uç uca eklediğimizde kenarları (a-b) ve (a+b) olan kusursuz yeni bir dikdörtgen elde ederiz. Alan değişmediği için eşitlik kanıtlanır.
